# Build Reliable Claude Code Workflows with Guardrails and Tests

This folder is a static GitHub Pages slide deck with 60 slides for a 3-hour training session.

## Structure

- Slides 1-20: reliable workflow foundations, specs, permissions, hooks, tests, and review.
- Slides 21-40: Skills, MCP, plugins, tool boundaries, and extension governance.
- Slides 41-60: compound engineering, agent dreaming, examples, capstone, and rollout.

## Diagrams

- Rendered slide diagrams live in `diagrams/svg`.
- Editable Excalidraw source files live in `diagrams/excalidraw`.
- Compact generation specs live in `diagrams/specs`.

## Publish on GitHub Pages

1. Copy these files to a GitHub repository.
2. Keep `index.html`, `styles.css`, and `script.js` at the repository root, or place the folder under `/docs`.
3. In GitHub, open **Settings > Pages**.
4. Choose the branch and folder that contain `index.html`.
5. Save. GitHub Pages will publish the deck.

## Presenting

- Use the **Back** and **Forward** buttons.
- Keyboard shortcuts also work: left/right arrows, Page Up/Page Down, Home, and End.
- Press `N` to toggle speaker notes.
- Add `?notes=1` to the URL to show notes by default.
